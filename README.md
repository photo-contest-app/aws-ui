# Photo Contest Frontend
Modern React application for the Photo Contest platform with AWS infrastructure.
## Features
- 📸 **Browse & Vote** - View photos one by one and vote YES or NO
- 🖼️ **Submit Photos** - Upload your best photo for the monthly contest
- 🏆 **Results** - See previous month's winner
- 🔐 **Authentication** - Secure login/register with AWS Cognito
- 📱 **Responsive Design** - Works on all devices
- ⚡ **Fast & Modern** - Built with Vite and React 19
## Setup
### 1. Install Dependencies
\`\`\`bash
npm install
\`\`\`
### 2. Configure API URL
Update \`.env\` with your backend API Gateway URL.
### 3. Run Development Server
\`\`\`bash
npm run dev
\`\`\`
## Deployment
\`\`\`bash
npm run build
npm run deploy
\`\`\`
